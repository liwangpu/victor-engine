动态引擎本质上是基于schema动态渲染的组件,包含插件机制,组件市场,组件数据采集,校验,联动以及权限等等
        表单设计器是引擎设计时与运行时涵盖的功能点


引擎分为渲染器和设计器两部分，独立工作
        设计器可视化生成schema,渲染器动态解析schema
        引擎以组件为单元,只有容器和非容器组件之分
        插件机制装配
        每个插件包含运行时组件本身和设计时的配置面板两部分

组件设计时与运行时隔离
        UI与数据隔离
        数据与副作用隔离（检验，填充）,事件驱动副作用
        副作用通过监听者形式进行管道处理变更
